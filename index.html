Large Format High Speed 3D Printer

This project involved designing and building a large format 3D Printer from scratch. It is an ongoing project, as I tend to add new features and upgrades every few months to it

Uses CoreXY kinematics, which allows faster speeds printing speeds by reducing the moving mass as the XY motors are stationary

Features: 410x410x400mm build area, Core XY kinematics, full enclosure, Duet 2 Wifi 32 bit ARM based controller, BMG direct drive extruder, independent dual Z axis (soon to be upgraded to 3)

The project involved a lot of CAD Design, and various method of manufacturing, such as 3D printing, laser cutting and CNC machining. Programming was done in Gcode

Currently working on upgrading the Z axis with 3 individually driven stepper motors to do true bed leveling, and working on a new extruder assembly that would allow flexible filaments to be printed better

AlcoLock IOT Device

This project involved designing, building and programming an IOT device, and was part of group Embedded Systems module coursework

Alcolock block's a pre-set list of websites when the user is drunk to prevent unnecessary purchases or drunk texts

The device consists of an Raspberry Pi Zero, an alcohol vapour sensor, an airflow sensor, and an i2c ADC. The reasings are sent to a desktop application via MQTT

The desktop application blocks sites based on the level of alcohol in the user's breath, and contains additional features such as graphs or automatic emails

Both the Alcolock device and desktop GUI were written in Python

Long Range Electric Longboard for Commuting

Capable of a speed of 48 km/h and a range of 55km, using 2x 2500W brushless 6355 motors, open source Electronic Speed Controller, and a flexible modular 900Wh battery with active management system capable of delivering continuous 160A.

As many parts as possible were built by me from scratch. I pressed the maple deck on the molds that I made, and then I made slots for the battery in the deck on a CNC router, I welded the battery pack out of 18650 cells and PCB modules I designed, I designed and made a 3D printed enclosure reinforced with fibreglass, etc.

Still in regular use, and required almost no maintenance in over 10000 miles since built.
